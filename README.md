# Mental Health Social Media Prediction

## Dataset
dataset pulled from [LLNL Jan 2026 Datathon](https://data-science.llnl.gov/latest/llnl-datathon)

## Methods
- Data preprocessing
- Feature encoding
- LinearSVC with hyperparameter tuning

## Results
it does good ~97% accuracy (f1 macro)
